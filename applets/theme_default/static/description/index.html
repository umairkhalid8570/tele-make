FAQs

Can this theme be used with Tele Enterprise?

Yes, this theme is tested with Tele Community as well as Enterprise Edition. for all the mentioned versions. We guarantee its compatibility for Tele's default addons for both CE and EE except any third party modules/apps/themes

Multi Website Feature is available with this theme?

Yes, Tele's Multi website feature is compatible with this Theme. You can configure this theme in multiple websites in the same database.

How can I do customization in this theme?

Our theme is customizable by using in-built building blocks and snippets. There are a number of headers, footers, pages to make a nice piece of website for your business. And in case you want any assistance in that, feel free to contact our technical team in email: support@tele.studio

Can I use demo data with this theme?

This demo data you see in the Live Demo is just for showing purposes. It's not given when you purchase the theme.

Are these images in your theme reusable?

No, the images loaded in the theme are copyrighted. In case you want to generate the images related to your business and website, feel free to contact us at our email: support@tele.studio

Is this theme resellable?

No. You can not resell this theme nor can you distribute it without the written consent of Tele INC.

Do I get a free upgrade for this theme if I have purchased it in the past?

Yes, we do. You get a free upgrade for the theme for the same Tele version. You can compare the changelog file with your installed theme version and download the latest version in case you're using the older version.

Moreover, if you want any feature which is not needed in even the latest theme version, feel free to contact us at our email: support@tele.studio

If you are looking to download the theme for different Tele versions then please note that starting version 13, every version of the theme will be sold separately.
